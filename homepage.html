Welcome to Fytobase

Introduction

This website aims to be a resource for understanding the phytochemicals (non-nutrient constituents) in regular foods. These molecules can:

  • Contribute to the culinary properties we enjoy in our meals or take into account during our cooking
  • Have pharmacological properties which may prove useful for DMPK scientists
  • May have health benefits - many of which are intimately linked to their agricultural methods of production (e.g. organic) or cooking
  • Explain how our body interacts with xenobiotics from a more evolutionary perspective

The project is currently under construction, and is being used as a way of learning the following coding skills:

  • HTML
  • CSS
  • Javascript
  • Python
  • Git and Github
  • SQLite
  • Domain settings
